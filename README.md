# Tasteology by JCDG

## Overview
This project was created as part of a technical assessment. It demonstrates responsive layout implementation using modern CSS techniques such as CSS Grid. The goal of this project is to showcase clean structure, maintainable styling, and responsive design principles.

## Features
- Responsive layout
- CSS Grid implementation
- Reusable CSS variables
- Organized file structure

## Installation / How to Run
1. Clone the repository:
   git clone https://github.com/lauvedesigns/tasteology.git

2. Open `index.html` in your browser.

## Notes
- The layout is optimized for desktop and mobile screens.
- No external frameworks were used.
